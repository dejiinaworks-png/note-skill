# 正しいコラム構造テンプレート

## 必須構成要素（絶対に入れる）

### 1. 冒頭キャッチ（記事冒頭）
**注意：見出しには■を使用する**
```
「○○○○」

この言葉を信じて...（衝撃的な事実・数字）

[感情を揺さぶる導入]
```

### 2. 目次（冒頭キャッチの直後）
```
## 目次
1. [セクション1タイトル]
2. [セクション2タイトル]  
3. [セクション3タイトル]
```
**注意：目次は3項目が基本。まとめは含めない。4項目以上は多すぎる**

### 3. アクションプラン（記事中盤〜後半）
```
今すぐやるべき3つのアクション

まず今日中にやること。○○○を確認する。

次に、今週中にやること。○○○を実践する。

最後に、今月中にやること。○○○を最適化する。
```

### 4. CTA（記事下部）
```
あなたには2つの選択肢しかない

選択肢1：今まで通り、○○を続ける。

選択肢2：○○で、○○を実現する。

問題は、あなたがどちらを選ぶか。

[行動を促す最終メッセージ]

もし本気で○○したいなら、詳細はこちらをご覧ください。
→ https://example.com/lp
```

## 標準セクション構成（3-4セクション）
**重要：各セクションの見出しには■を付ける**

### セクション1：問題の本質
- 読者が直面している課題の明確化
- 表面的問題vs根本的問題の区別
- 現状維持のリスク

### セクション2：失敗パターンの解説
- よくある間違いの具体例
- 失敗する人の共通点
- 避けるべき行動・思考

### セクション3：正しいアプローチ
- 問題解決の基本概念
- 成功の原理・原則
- 考え方の転換ポイント

### セクション4：具体的実践方法
- ステップバイステップの手順
- 実行時の注意点
- 効果測定の方法

### セクション5：成功事例・応用例
- 実際の成功ストーリー
- 具体的な数字・結果
- 応用・発展のアイデア

### セクション6：よくある質問・トラブル対処
- 実践時の疑問点
- つまづきやすいポイント
- 解決策・代替案

### セクション7：まとめ・次のステップ
- 重要ポイントの再確認
- 段階的な実践プラン
- さらなる成長への道筋

## 文字数配分（5000文字の場合）

- **目次**: 100文字
- **導入**: 400-500文字（8-10%）
- **セクション1**: 600-700文字
- **セクション2**: 600-700文字
- **セクション3**: 700-800文字
- **セクション4**: 800-900文字
- **セクション5**: 600-700文字
- **アクションプラン**: 300-400文字
- **まとめ・CTA**: 400-500文字

## 実践性を高める要素

### 具体的なツール・リソース
- チェックリスト
- テンプレート
- 計算シート
- 参考リンク

### 段階別ガイド
- 初心者向け
- 中級者向け
- 上級者向け

### 期限・スケジュール
- 今日やること
- 今週やること
- 今月やること

## CTAの種類とパターン

### 1. 相談・問い合わせ誘導
「○○でお困りの方は、お気軽にご相談ください」

### 2. 資料ダウンロード
「詳しい資料を無料でダウンロードできます」

### 3. セミナー・イベント参加
「次回のセミナーでお会いしましょう」

### 4. サービス・商品紹介
「さらに詳しく学びたい方はこちら」

### 5. コミュニティ参加
「同じ志を持つ仲間と一緒に成長しませんか」

## 品質チェックポイント

### 構造チェック
□ 目次が設置されている
□ アクションプランが含まれている
□ CTAが明確に配置されている
□ セクション構成が論理的
□ 実践的な内容になっている

### 内容チェック
□ 読者の問題を明確にしている
□ 具体的な解決策を提示している
□ 実行可能なステップがある
□ 成功事例が含まれている
□ 次のアクションが明確

### 読みやすさチェック
□ 適切な見出し構成
□ 文章の長さが適切
□ 改行・段落分けが効果的
□ 重要ポイントが分かりやすい
□ 全体の流れが自然

このテンプレートに沿って作成することで、読者にとって価値があり、実践的で、行動を促すコラムを作成できます。
