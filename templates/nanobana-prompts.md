# NanoBanana2 プロンプトテンプレート集

## 1. バナー（アイキャッチ）

### 基本テンプレート
```
noteアイキャッチバナーを作成してください。
タイトル: [記事タイトル]
サイズ: 1280×670px（noteのOGP推奨サイズ）
スタイル: モダン・プロフェッショナル・インパクト重視
メインカラー: [ブルー/グリーン/オレンジ/パープル]系
背景: グラデーション（暗め）
テキスト配置: 中央・大きめフォント・白文字
サブテキスト: [サブタイトルや補足（任意）]
アイコン・イラスト: [テーマに合うアイコン（任意）]
```

### AI・テクノロジー系バナー
```
AIテクノロジー系のnoteアイキャッチバナー
タイトル: [記事タイトル]
コンセプト: 未来感・テクノロジー・信頼性
カラー: ディープブルー × シアン × ホワイト
背景: 暗いグラデーション＋回路/ドット模様
フォント: 太ゴシック・スッキリ
```

### 副業・お金系バナー
```
副業・収益化系のnoteアイキャッチバナー
タイトル: [記事タイトル]
コンセプト: 成功・希望・行動喚起
カラー: ゴールド × ネイビー or オレンジ × ホワイト
背景: 上昇グラフ or コイン・グラフィック
フォント: 力強いゴシック
```

## 2. フロー図

### 基本テンプレート
```
フローチャートを作成してください。
テーマ: [図解のテーマ]
ステップ数: [N]ステップ

ステップ1: [内容]
ステップ2: [内容]
ステップ3: [内容]
（以下続く）

スタイル: シンプル・矢印あり・ラウンド角
メインカラー: [ブルー/グリーン]系
背景: 白 or 薄いグレー
フォント: 丸ゴシック・読みやすいサイズ
```

### 3ステップ型（シンプル）
```
3ステップのシンプルなフロー図
ステップ1: [アクション] → [結果]
ステップ2: [アクション] → [結果]
ステップ3: [アクション] → [結果]
矢印: 右向き or 下向き
アイコン: 各ステップに関連アイコン付き
カラー: グラデーション（左から右へ色が変わる）
```

## 3. 比較図

### Before / After型
```
Before/After比較図を作成してください。
テーマ: [比較テーマ]

Before（左側）:
- [状態1]
- [状態2]
- [状態3]

After（右側）:
- [状態1]
- [状態2]
- [状態3]

スタイル: 2カラム・中央に矢印
Beforeカラー: グレー系（暗い印象）
Afterカラー: ブルーまたはグリーン（明るい印象）
```

### 比較表型
```
比較表を作成してください。
比較対象: [A] vs [B]

比較項目:
- [項目1]: A=[評価] / B=[評価]
- [項目2]: A=[評価] / B=[評価]
- [項目3]: A=[評価] / B=[評価]
- [項目4]: A=[評価] / B=[評価]
- [項目5]: A=[評価] / B=[評価]

スタイル: テーブル形式・シンプル
おすすめ表示: [どちらか]にハイライト
カラー: ヘッダーにアクセントカラー
```

## 4. まとめインフォグラフィック

### カード型
```
まとめインフォグラフィックを作成してください。
タイトル: 「[記事タイトル]のまとめ」
要点数: [N]個

要点リスト:
1. [要点1]
2. [要点2]
3. [要点3]
（以下続く）

スタイル: カード型・番号付き・アイコン付き
カラー: メインカラー + 白背景カード
レイアウト: 縦一列 or 2カラム
フォント: 丸ゴシック・大きめ
```

### チェックリスト型
```
チェックリスト形式のインフォグラフィック
タイトル: 「[タイトル]チェックリスト」
項目数: [N]項目

チェック項目:
□ [項目1]
□ [項目2]
□ [項目3]
（以下続く）

スタイル: チェックボックス付き・シンプル
カラー: グリーン系（達成感）
用途: note記事のまとめ・SNSシェア用
```

## 5. データ・グラフ系

### 成長グラフ
```
成長グラフを作成してください。
タイトル: [グラフタイトル]
X軸: [期間（月/週/日）]
Y軸: [指標名]
データポイント: [N]点

変化の特徴: [右肩上がり/急上昇/安定成長]
強調ポイント: [N番目のポイントをハイライト]
カラー: ブルー系グラデーション
スタイル: 折れ線 or 棒グラフ・シンプル
```

## プロンプト活用Tips

1. **具体的な数字を入れる** → より的確な図解になる
2. **カラーテーマを統一する** → 記事全体に一体感
3. **ターゲット読者を意識する** → 「初心者向け=シンプル」「プロ向け=詳細」
4. **アイコンを指定する** → 視認性が上がり印象に残る
5. **白背景を基本にする** → noteの背景（白）に馴染む
