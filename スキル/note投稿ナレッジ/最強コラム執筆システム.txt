## 第2部：感情操作マスターテンプレート

### 冒頭衝撃パターン（必殺技）

**パターンA：常識完全否定**
```
「○○すれば成功する」

この常識を信じた99%の人が、人生を棒に振っています。

なぜなら、これは○○業界が仕掛けた壮大な嘘だから。

今日は、あなたが騙され続けている真実を暴露します。
```

**パターンB：数字の暴力**
```
月収30万円のサラリーマンが、3ヶ月で年収3000万円。

嘘だと思いますか？

でもこれ、実際に起きた話です。しかも使ったのは、あなたも知っている「あの方法」だけ。

ただし、99%の人はこの方法を間違って使っています。
```

**パターンC：業界暴露**
```
「お客様は神様です」

この言葉を真に受けた会社の87%が、5年以内に倒産しています。

○○業界で20年働いてきた私が、この業界の闇を暴露します。

知らないと、あなたも確実に騙されます。
```

## 第3部：トミー語調完全再現システム

### 基本語調パターン

**断定調（権威性演出）**
- 「〜です」「〜である」「〜に決まっている」
- 「間違いありません」「確実に言えます」
- 「私が保証します」

**共感調（距離感調整）**
- 「〜ですよね」「〜じゃないですか」
- 「分かります、その気持ち」
- 「私も同じでした」

**挑発調（感情喚起）**
- 「いい加減、気づいてください」
- 「そんな甘い考えでは」
- 「現実を見てください」

### 文章リズムの黄金法則

**短文で衝撃を与える**
```
これが現実です。

厳しいですが、事実です。

あなたには、2つの選択肢しかありません。
```

**長文で感情を揺さぶる**
```
私がこの方法を知ったのは、人生のどん底にいた時でした。借金は2000万円、家族にも見捨てられ、もう死ぬことしか考えられませんでした。
```

**疑問文で思考を促す**
```
なぜ、こんなことが起きるのでしょうか？

答えは簡単です。

○○業界が、意図的に隠しているからです。
```

## 第6部：品質保証チェックシステム

### 必殺フレーズチェック
□ 冒頭で常識を完全否定している
□ 「99%」「95%」等の圧倒的多数への言及
□ 業界の嘘・欺瞞を暴露している
□ 読者の甘い考えを粉砕している
□ 成功者vs失敗者の残酷な対比がある

### 感情操作チェック
□ 冒頭で読者を殴っている
□ 各セクションで感情の起伏を作っている
□ 危機感・恐怖感を適切に演出している
□ 希望・期待感も提示している
□ 最後に決断を迫っている

### トミー語調チェック
□ 断定調で権威性を演出している
□ 適度に共感調で距離感を調整している
□ 短文と長文のリズムが効いている
□ 読者への問いかけがある
□ 「あなた」を多用せず主語を工夫している

このシステムを使えば、読者を完全に支配する最強のコラムが作成できます。
